Grocery List Manager

A simple and responsive grocery list web application built using React.js. This application allows users to add, delete, search, and manage grocery items dynamically. It demonstrates core React concepts such as component-based architecture, state management, and list rendering.

Features

• Add new grocery items
• Delete existing items
• Check boxe to mark items as completed
• Search grocery items
• Simple user interface
• Dynamic list rendering using React

Tech Stack

• React.js
• JavaScript (ES6+)
• HTML5
• CSS3

Learning Outcomes
• Understanding of React functional components
• Usage of useState hook
• Handling events in React
• Conditional rendering
• Working with lists and keys
• Basic state management
